Sintaxis Básica

Conceptos fundamentales de Python

Variables
# Variables en Python nombre = "Gabriel" edad = 25 promedio = 85.5 es_estudiante = True # Imprimir variables print(f"Hola, soy {nombre}")
Tipos de Datos
# Tipos básicos texto = "Hola mundo" # str numero = 42 # int decimal = 3.14 # float es_valido = True # bool # Colecciones lista = [1, 2, 3] # list tupla = (1, 2, 3) # tuple
Condicionales (if)
nota = 75 if nota >= 70: print("Aprobado") elif nota >= 50: print("Aceptable") else: print("Reprobado")
Bucles (for/while)
# Bucle for for i in range(5): print(i) # Bucle while contador = 0 while contador < 5: print(contador) contador += 1
Funciones
def saludo(nombre): return f"Hola, {nombre}" # Llamar función mensaje = saludo("Gabriel") print(mensaje)
Entrada/Salida
# Entrada de usuario nombre = input("¿Tu nombre?") # Entrada con tipo edad = int(input("¿Tu edad?")) # Salida formateada print(f"Tienes {edad} años")
